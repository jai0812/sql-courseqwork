### SQL 01 fundamentals - Relational models

`This lab demonstrates core SQL concepts through hands-on
table creation, data manipulation, constraints, and
anomaly analysis.`

## Concepts Covered

- CREATE and DROP TABLE
- INSERT, SELECT, UPDATE, DELETE
- NOT NULL and PRIMARY KEY constraints
- Targeted queries using primary keys
- Insertion and deletion anomalies
- Importance of data integrity in relational design
  
## Key Learning Outcomes
- Understanding why constraints matter
- Seeing how poor table design leads to data anomalies
- Practicing safe and precise SQL updates and deletions

## Tools & Technologies
- SQL (ANSI-compliant)
- Relational Database Concepts

practice designed to build strong database fundamentals.
